# Minecraft服务器状态查询页面

这是一个用于实时查询Minecraft服务器状态的Web页面，支持多服务器监控、主题切换和自动刷新功能。

## 功能特点

### 核心功能
- ✅ **实时服务器状态查询**：显示服务器在线/离线状态
- ✅ **多服务器支持**：可同时监控多个Minecraft服务器
- ✅ **自动刷新**：定时自动更新服务器状态
- ✅ **主题切换**：支持亮色/暗色模式切换

### 显示信息
- 🎮 **服务器基本信息**：IP地址、端口、版本
- 👥 **玩家信息**：在线人数、最大人数、玩家列表
- ⏱️ **性能数据**：延迟、协议版本
- 🎨 **服务器图标**：显示服务器MOTD和图标

### 用户体验
- 📱 **响应式设计**：完美适配桌面、平板和手机
- 🎨 **现代化界面**：卡片式设计，视觉效果优雅
- 🔄 **自动排序**：在线服务器自动显示在离线服务器上方
- 💾 **主题记忆**：自动保存用户的主题偏好

## 快速开始

### 1. 克隆或下载项目
将项目文件下载到本地目录。

### 2. 配置服务器列表
打开 `index.html` 文件，找到 `CONFIG` 对象，修改 `SERVERS` 数组以添加或修改要监控的服务器：

```javascript
const CONFIG = {
    // 服务器列表配置
    SERVERS: [
        { ip: 's6.ungc.com.cn', port: '15120', name: '童话镇' },
        { ip: 'mc.fnzjaq.me', port: '25565', name: '你也来🐱🐟么' },
        { ip: 'okkmc.cn', port: '32230', name: 'OKKMC' }
        // 可添加更多服务器
    ],
    // ... 其他配置
};
```

### 3. 启动应用
使用任意Web服务器打开 `index.html` 文件即可运行。

#### 推荐方法：
- **使用Python内置服务器**：
  ```bash
  python -m http.server 8000
  ```
  然后在浏览器中访问 `http://localhost:8000`

- **直接打开**：双击 `index.html` 文件直接在浏览器中打开

## 配置选项

### 服务器配置
- `ip`: 服务器IP地址
- `port`: 服务器端口（默认25565）
- `name`: 服务器显示名称

### 应用配置

在 `CONFIG` 对象中可以修改以下配置：

- `API_BASE`: 用于查询服务器状态的API地址
- `CACHE_DURATION`: 缓存持续时间（毫秒）
- `AUTO_REFRESH_INTERVAL`: 自动刷新间隔（毫秒）
- `MAX_RETRIES`: API请求最大重试次数

## 技术栈

- **前端框架**：纯HTML/CSS/JavaScript（无需依赖）
- **API**：使用 [MOTD API](https://motd.minebbs.com/api/status) 查询服务器状态
- **设计**：现代化响应式设计，支持主题切换

## 使用说明

### 主题切换
点击页面右上角的🌙/☀️图标可切换亮色/暗色模式。

### 查看详细信息
服务器卡片支持点击交互，悬停时会有轻微的视觉反馈。

### 刷新数据
页面会自动定期刷新服务器状态。

## 常见问题

### Q: 服务器状态显示错误？
A: 请检查服务器IP和端口是否正确，确保服务器可以正常访问。

### Q: 页面加载缓慢？
A: 可能是网络问题或API响应延迟，请稍后重试。

### Q: 如何添加更多服务器？
A: 在 `CONFIG.SERVERS` 数组中添加新的服务器对象即可。
